Projects

Pomodoro timer

The Pomodoro Technique is a time management method developed by Francesco Cirillo in the late 1980s. The technique uses a timer to break down work into intervals, traditionally 25 minutes in length, separated by short breaks. In this project, I made simple web application that helps you to organize your time following pomodoro rules.

Static site generator

Static Site Generator is a simple web application that makes your job easyer, and saves your time at starting new project. At the moment, JavaScript generates Title, Image, Description and Background color, but I'll try in future to expand this code and make it more interesting, and usable in practice.
